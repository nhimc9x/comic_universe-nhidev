This is a comic reading website that I created during the process of learning to call APIs and using Redux.


## Built With

This project was built using these technologies.

- Vite
- React.js
- TailwindCSS
- CSS3
- Swiper
- Axios
- Redux toolkit
- Rest Apis
- VsCode
- Vercel

**🌗 Dark Mode**

**📚 Filter Comics**

**🔎 Search Comics**

**️🏅 Ranking Comics**

**🖼️ 3D Effect**

**📱 Fully Responsive**
